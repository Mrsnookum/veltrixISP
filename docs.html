System Documentation

Welcome to Veltrix. This guide will walk you through configuring your MikroTik router and setting up automated M-Pesa billing for your Wi-Fi Hotspot.

Hardware Requirements

To ensure seamless operation with Veltrix ISP, you will need the following hardware:

  • MikroTik Router: RouterOS v6 or v7 (v7 recommended). Popular models include hEX (RB750Gr3) for small setups or RB4011 for medium to large networks.
  • Active Internet Connection: A stable WAN connection from a primary ISP (e.g., fiber, dedicated wireless, or 4G/5G modem).
  • Configuration Device: A laptop or PC with an Ethernet port to run WinBox for the initial setup.
  • Access Points (Optional but Recommended): External access points connected to your MikroTik to broadcast the Wi-Fi signal to your clients (e.g., Ubiquiti, TP-Link, or Cambium).
Phase 1

Physical Setup & First Login

Before configuring software, ensure your hardware is properly connected.

  • Internet In: Plug the cable from your ISP (Safaricom, Zuku, etc.) into Port 1 of the MikroTik.
  • PC Connection: Plug your laptop into Port 2 using an Ethernet cable.
  • WinBox: Open WinBox, click the "Neighbors" tab, click your router's MAC address, and login with Username: admin and leave the password blank.

Phase 2

Providing Internet Access to the Router

The MikroTik needs an internet connection before it can serve your clients. This is usually done via a DHCP Client.

  1. In WinBox, go to IP > DHCP Client.
  2. Click the + (Add) button.
  3. Select ether1 (or whichever port your main internet line is plugged into) as the interface.
  4. Ensure "Use Peer DNS" and "Use Peer NTP" are checked.
  5. Click Apply and OK. You should see it acquire an IP address and show a status of "bound".
Note: If your ISP provides a static IP or uses PPPoE, configure that connection under IP > Addresses and IP > Routes, or using the PPPoE Client interface instead.

Phase 3

Creating the Hotspot Server

Now we create the network that will serve the captive portal to your clients.

  1. First, assign an IP address to the port serving your local network (e.g., ether2). Go to IP > Addresses, click +, enter an address like 10.0.0.1/24, and select ether2.
  2. Go to IP > Hotspot.
  3. Click the Hotspot Setup button (do not click the + button).
  4. Follow the setup wizard:
    • Hotspot Interface: Select ether2 (or your bridge if you have multiple ports).
    • Local Address of Network: Keep the default (e.g., 10.0.0.1/24).
    • Address Pool: Keep the default.
    • Select Certificate: Choose "none".
    • IP Address of SMTP: Keep 0.0.0.0.
    • DNS Servers: Add 8.8.8.8 and 8.8.4.4.
    • DNS Name: Enter a domain name for your login page (e.g., login.veltrix). This must match the DNS name in your Veltrix dashboard later.
    • Name of Local Hotspot User: Leave default.

Phase 4

Linking the Billing API

This is the most critical step. It allows the Veltrix cloud dashboard to automatically authenticate users and expire them when their time is up.

  1. Go to your Veltrix Admin Dashboard and open the MikroTik Tab.
  2. Enter your preferred API Username and Password.
  3. Click Save & Generate Script.
  4. Copy the generated green code block.
  5. In WinBox, click New Terminal on the left menu, paste the code, and press Enter.
/user add name="api_user" group=full password="your_password"
/ip service set api disabled=no port=8728
/ip service set api-ssl disabled=no port=8729
/ip cloud set ddns-enabled=yes
Phase 5

Uploading the Portal Redirect

You must replace the default MikroTik login page so users are directed to your custom Veltrix billing portal.

Tip: In your Admin Dashboard (Setup Guide tab), there is a button to dynamically download your specific login.html file. Download it there.

In WinBox, click Files. Locate the hotspot folder. Drag and drop your downloaded login.html directly into this folder, overwriting the old one.

Configuring PayHero (M-Pesa)

Veltrix natively supports PayHero for automated M-Pesa STK Pushes.

  1. Create an account at PayHero Kenya.
  2. Link your Paybill or Till Number.
  3. Navigate to Developers > API Keys.
  4. Copy your Channel ID and your base64 encoded Basic Auth Token.
  5. Paste these into the API Settings tab in your Veltrix Admin Dashboard.

Need a Custom Payment Gateway?

While PayHero is our default automated gateway, we understand that established ISPs may prefer direct Daraja API integrations, Stripe, Flutterwave, or custom bank APIs to avoid third-party transaction fees.
